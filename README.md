A professional, enterprise-grade Software Requirements Specification (SRS) and interactive documentation platform for the **FLEXPro ERP** ecosystem. This project provides a comprehensive blueprint for a scalable, reliable, and high-performance ERP system designed for modern business operations.

---

## 🚀 Overview

FLEXPro ERP is designed to streamline complex business processes across multiple departments, including Sales, Inventory, CRM, and Finance. This documentation repository serves as the single source of truth for the system's architecture, data models, and functional requirements.

### Key Features
- **Enterprise Architecture**: Built for scalability with a multi-branch, real-time synchronization engine.
- **Comprehensive SRS**: Detailed requirements mapping for all core business modules.
- **Technical Blueprints**: Interactive Database Schemas (ERD), System Workflows, and UI Mockups.
- **Bilingual Support**: Full English and Arabic localized content with RTL support.
- **Professional Export**: Ready-to-use print and export functionality for stakeholder reporting.

---

## 🏗️ Technical Stack

- **Core**: React 19, TypeScript
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS 4, Lucide React (Icons)
- **Animations**: Motion (formerly Framer Motion)
- **Content**: Markdown-driven documentation with Remark-GFM

## 🛠️ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/skcc-dev/FLEXPRO-Doc.git
   cd FLEXPRO-Doc
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
